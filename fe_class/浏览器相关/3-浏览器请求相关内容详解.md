为什么cdn 域名和业务域名不相同？

// www.baidu.com 业务域名
// cdn.baidu-aa.com cdn 域名

1. 安全问题：（不同源在当前浏览器同源策略下就不能携带cookie)
2. cdn request header 会携带 cookie. 带宽消耗、流量消耗
3. 并发请求数：（http/1.1 限制你同源统一域名同一时间只能发送一定的请求，cdn资源会阻塞业务数据资源）

解决方案
